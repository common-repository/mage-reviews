Add Review Rating fields to comment form on posts / custom post types with stars or custom icon, rate count and many more options with SEO markup.

== Description ==

By default, Mage Reviews adds a 5 Star Rating field to comment forms on posts and displays them on Author comments. Easily set the **Rating Count**, **Rating Icons**, **Custom Post Types** to add rating fields to, and much more.

= Main Features Include =
* Add Rating option to Comment forms on any Post Types.
* Show Average Rating to single posts and archives.
* Choose from dozens of Rating Icons or Upload your own.
* Display Recent Reviews with simple Widget or Shortcode.
* SEO Rich Snippet Markup with Schema for Google Results.
* BuddyPress Members and Logged In User/Author Compatible.

= User Rating Fields and Icon options =
1. Choose one or multiple post types to display rating fields on.
1. Customize default icon and hover icon state.
1. Set lowest and highest rating option. 
1. Choose from available icons or upload custom icon.
1. Option to require a rating to submit comment.
1. Restrict users to one submission per post.


= Instructions =
All Instructions can be found from the plugins "Help" tab on the WP admin.

== Installation ==

1. Extract and Upload Content of `mage-reviews.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR

1. Search 'mage reviews' through the 'Plugins' menu in WordPress
2. Install 'Mage Reviews' from search results.
